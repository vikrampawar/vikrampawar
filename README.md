## Hi, I'm Vikram

Software engineer in London. I build tools and applications around AI, with a focus on Claude and the Model Context Protocol (MCP) ecosystem.

### What I'm working on

- **[BioVibing](https://github.com/vikrampawar/biovibing-mcp-app-claude-code-london-03)** — Health dashboard MCP App that renders Oura Ring biometrics inside Claude.ai. Built with Skybridge for the Claude Code London hackathon.
- **[Claudia](https://github.com/vikrampawar/claudia)** — GUI app and toolkit for Claude Code. Custom agents, interactive sessions, background agents.
- **[Claude Code Training](https://github.com/vikrampawar/claude-code-training)** — Slides and demos for a training course on Claude Code.
